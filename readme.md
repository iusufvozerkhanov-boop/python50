# Testwork Groupe 2

Тестовое веб-приложение TODO на **Flask**, реализованное с использованием **SQLite**.

## Возможности (ТЗ)
- ✅ Добавление задач через форму
- ✅ Сохранение задач в SQLite
- ✅ Отображение всех задач на главной странице
- ✅ Кнопка «Удалить» рядом с задачей
- ✅ Возможность отметить задачу выполненной (зачёркивание)
- ✅ Минималистичная стилизация (CSS)
- ✅ Мини-инструкция в README

## Стек
- Python 3
- Flask
- SQLite
- HTML / CSS

## Запуск проекта

1. Установить Flask:

```pip install flask```

2. Запустить приложение:

```python main.py```


3. Открыть в браузере:

```http://127.0.0.1:5000```

4. Структура проекта

```
testwork_groupe2/
├── main.py                  # основной файл Flask-приложения
├── data.py                  # логика работы с базами данных
├── tasks.db                 # база данных активных задач
├── completed.db             # база данных выполненных задач
├── templates/
│   ├── base.html            # базовый HTML-шаблон
│   ├── index.html           # главная страница (активные задачи)
│   └── completed.html       # страница выполненных задач
├── static/
│   └── style.css            # стили
└── README.md                # описание проекта

```

5. Примечание

```База данных SQLite создаётся автоматически при первом запуске```
